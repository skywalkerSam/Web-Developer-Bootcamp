
## Using [Firebase](https://firebase.google.com/)

```shell
npm i -g firebase-tools
```

```shell
firebase login
```

```shell
firebase init hosting
```

**NOTE**: Select the `build` folder as the public directory.

```shell
npm run build
```

```shell
firebase deploy
```
